### Task 2 - Landing Page
- **Folder:** `Task-2-LandingPage`
- **Description:** A responsive landing page named **Orbit** designed with Tailwind CSS and interactive elements.
- **Technologies Used:** HTML, Tailwind CSS, JavaScript
- **Features:**
  - Modern hero section with call-to-action buttons
  - Testimonials section with carousel/slider effect
  - Mobile-friendly layout
- **Live Demo:** [Orbit] (https://aaiswaryapm.github.io/CODSOFT/Task-2-LandingPage/)
- **Task README:** [`Task-2-LandingPage/README.md`] (Task-2-LandingPage/README.md)

---

### Task 3 - Calculator Web Application
- **Folder:** `Task-3-Calculator`
- **Description:** A feature-rich calculator named **CalcPro** which performs basic arithmetic operations with a user-friendly interface.
- **Technologies Used:** HTML, CSS, JavaScript
- **Features:**
  - Performs addition, subtraction, multiplication, and division
  - Theme toggle (dark/light mode)
  - History panel to track previous calculations
  - Automatically rounds off decimals
  - Error detection for invalid operations (e.g., division by zero)
- **Live Demo:** [CalcPro] (https://aaiswaryapm.github.io/CODSOFT/Task-3-Calculator/)
- **Task README:** [`Task-3-Calculator/README.md`] (Task-2-Calculator/README.md)
